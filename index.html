Method

This is a practical local benchmark, not a universal claim about any package or language.

How to rerun

From this directory, run ./run_benchmarks.sh. Set BENCH_ITERATIONS=12 to collect more repetitions.

Matrix is used for dense matrix multiplication when installed. data.table is used for sorting and group-by variants when installed.
